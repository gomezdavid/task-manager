## Task Manager

Fullstack task management application built with **.NET 8**, **React**, and **SQL Server**.  
This project allows users to create, manage, and track tasks efficiently through a modern web interface.

---

## Features

- Create, update, and delete tasks
- Mark tasks as completed
- Filter and organize tasks
- RESTful API 
- Responsive frontend built with React

---

## Tech Stack

### Backend
- .NET 8 (ASP.NET Core Web API)
- Entity Framework Core
- SQL Server

### Frontend
- React
- CSS 

### Database
- SQL Server
